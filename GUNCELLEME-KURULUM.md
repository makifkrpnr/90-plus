# 90+ v5 — Render Güncellemesi

1. GitHub Desktop ile mevcut `90-plus` reposunu aç.
2. Bu paketin içindeki bütün dosyaları repo köküne kopyala ve eskilerinin üzerine yaz.
3. Eski `package-lock.json` varsa sil.
4. Hazır seslerini `assets/audio/` altında doğru klasörlere yerleştir.
5. GitHub Desktop'ta `90+ v5 mobile arena` mesajıyla Commit ve **Push origin** yap.
6. Render otomatik başlamazsa: **Manual Deploy → Clear build cache & deploy**.
7. Logda `socket.io@4.8.1`, ardından `90+ çalışıyor` mesajını kontrol et.
8. `https://SERVISIN.onrender.com/health` adresi `{"ok":true}` dönmelidir.
